Over mij

Nieuwsgierig,
gedreven & creatief.

Als student aan Arteveldehogeschool leer ik elke dag bij — van frontend tot backend, van databases tot mobiele apps. Buiten mijn studies vind je mij op het sportveld, diep in een muziekplaylist of bezig met creatieve projecten.


Mijn meerwaarde? Ik benader elk project als eindgebruiker: de code moet niet alleen werken, ze moet ook aangenaam aanvoelen. Ik ben leergierig, werk graag in team en lever altijd nette, leesbare code op.
